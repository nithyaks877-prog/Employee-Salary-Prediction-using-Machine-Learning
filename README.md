# Employee Salary Prediction using Machine Learning

## 📌 Project Overview

This project predicts the salary of newly hired employees based on their professional and educational attributes using Machine Learning techniques. The model helps estimate expected salaries by learning patterns from historical employee data.

This project demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and prediction.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Feature Scaling
- Machine Learning Model Training
- Model Evaluation
- Salary Prediction for New Employees
- Visualization of Results

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/nithyaks877-prog/Employee-Salary-Prediction.git
```

Move into the project directory:

```bash
cd Employee-Salary-Prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Jupyter Notebook:

```bash
jupyter notebook
```
